# savvital_assesment
Assesment Directory containing two main subfolder named **task1** and **task2**. 

## Task 1: Estate Planning AI Automation

- **task1** contains a for named **workflow** in which there a JSON file for workflow you can import it directly to n8n.
- Just one requirement I have used Groq Free API you have to set credentials for that. 
- As you press the **Execute Workflow** a form will open for submission where you can enter the information manually or you can upload (CSV, JSON, XLSX) file containing multiple rows of data.
- After form submission you will directly get the Result screen in which you can see your output. 
- There are some other supporting files in the **task1** with name (clients.csv, clients.json and evidence_log.docx) in docx file you can see evidence logs or you can directly see it in n8n executions. 
- There is a file generate_evidence_log.py it just take json object from file and make out a structured document  it just requrie ```pip install python-docx``` 
- Also there is a README.md file which you overall overview of the task perfomed.
